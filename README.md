# Customer Churn Prediction System

This project predicts whether a customer will churn using Machine Learning.

## Features
- ML models (XGBoost, Neural Network)
- Streamlit UI
- Agentic AI (suggests actions)

## Tech Used
- Python
- Scikit-learn
- XGBoost
- Streamlit

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run app:
   streamlit run app.py

## Output
- Predicts churn
- Suggests actions (offer/discount)
